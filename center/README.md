# Enernet Center

> 에너넷 전사 데이터팀의 업무 자동화 개발 및 분석 센터

Center는 반복적인 현업 데이터 업무를 분석 가능한 구조로 정리하고, 전처리·판정·검토·집계·대시보드까지 연결하는 내부 자동화 프로젝트 모음입니다.

## 하위 기능

| 기능                        | 설명                                                                                                        | 상세 문서                                |
| --------------------------- | ----------------------------------------------------------------------------------------------------------- | ---------------------------------------- |
| 서비스오더 · 오생성(Error) | 서비스오더 Excel의 소분류와 내역 불일치를 후보·자동 오생성으로 분류하고, 관리자 검토 및 BI 대시보드로 운영 | [README](./service_order/error/README.md) |

## 공통 원칙

- 개인정보와 원본 업무 데이터는 Git에 저장하지 않습니다.
- 반복 데이터는 규칙과 승인 근거로 축적하되, 자동 확정은 보수적으로 적용합니다.
- 집계 수치, 관리자 검토 결과, 대시보드는 같은 데이터 기준을 사용합니다.
- 각 기능은 독립적으로 확장할 수 있도록 화면·분석·저장소·규칙을 분리합니다.
